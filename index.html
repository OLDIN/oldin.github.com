Этапы создания сайта:

Разработка дизайна.

Конечным продуктом, этого этапа, являются макеты - грубо говоря, обычные картинки. Если заказчика устраивает внешний вид макетов, то осуществляется переход ко второму этапу создания сайта.

Верстка и web-программирование.

Верстка - процесс формирования веб-страниц из макетов. Осуществляется обеспечение совместимости со множеством браузеров (кроссбраузерность). Разработка программной основы, используя CMS (систему управления контентом), основываясь на подготовленные части сайта и элементы интерфейса.

Размещение сайта в сети.

Для того чтобы сайт был доступен для всех, его необходимо выложить на сервер. На этом этапе осуществляется регистрация доменного имени и загрузка файлов сайта на сервер хостинга. Мой хостинг-провайдер BeGet.ru
