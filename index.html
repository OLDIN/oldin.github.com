CSWORLDDROP

Перед мною стояла задача разработать проект открытия кейсов для [CS:GO].
Проект должен содержать на главной странице перечень кейсов, которые можно открыть.
После перехода в один из них будет выведен список скинов [CS:GO], которые можно будет выиграть.
При нажатии на кнопку "открыть" крутится рулетка и выпавший скин можно будет продать внутри проекта или вывести в STEAM.
Для проведения платижей была прикручена robokassa.

Так же нужно было разработать второстепенные страницы, а именно: FAQ | гарантии, ТОП, отзывы участников, контакты, новости, пользовательское соглашение, обратная связь.

Через некоторе время, по просьбе заказчика, был добавлен ряд языков для интерфеса проекта, а именно: немецкий, французский, английский.

Используемые языки и технологии: CSS, HTML, PHP, JavaScript, jQuery, Ajax
